Detailed Description

template<typename T, std::size_t ELEMENTS>
class fixed_allocator< T, ELEMENTS >

Discription of the "Fixed Allocator" class.

This class will allow you to pre-allocate memory for data in such containers that do not have to do this using standart methods.

Template Parameters
T- data types.
ELEMENTS- the size of memory to reserv.

Detailed Description

template<typename T, std::size_t CAPACITY>
class chunk_list< T, CAPACITY >

Discription structure to the allocated memory.

This is the buffer that represents the simple doubly linked list.

Template Parameters
T- the type of the data in the cells.
CAPACITY- number of memory cells of a given type.

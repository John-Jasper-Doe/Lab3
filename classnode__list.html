Detailed Description

template<typename T, typename A = std::allocator<node<T>>>
class node_list< T, A >

Discription of the container for working with a single-linked list.

Template Parameters
T- the type of variable stored in the node.
A- allocator, memory manager for working with container. Default on std::allocator.
